## Framework de Identificação de Momento Viral (Vitor Viral)

Um momento viral é uma janela de 15–60 segundos que contém:

### Critérios de Seleção (Score por Critério)
| Critério | Peso | Como identificar |
|---------|------|-----------------|
| Informação surpreendente / contraintuitiva | 30% | Palavras: "Mas...", "Na verdade...", "Você não sabe que..." |
| Curiosity gap presente | 25% | Pergunta sem resposta imediata, setup de revelação |
| Energia emocional alta | 20% | Tom de voz elevado, exclamações, ritmo acelerado |
| Ponto de entrada limpo | 15% | Começa sem depender de contexto anterior |
| Loop possível | 10% | Final conecta ao tema do início |

### Fórmula de Score
```
Score = (Surpresa × 0.30) + (Curiosidade × 0.25) + (Energia × 0.20) + (Entrada Limpa × 0.15) + (Loop × 0.10)
```
Score mínimo para selecionar: 0.65 / 1.00

## Framework de Edição (Carlos Corte)

### Sequência de Produção
1. **Extrair clipe** com 2–3 segundos de buffer antes e depois do timestamp
2. **Identificar o melhor frame de entrada** — deve criar curiosidade imediatamente
3. **Adicionar hook visual** nos primeiros 2 segundos (texto em overlay: máx. 8 palavras)
4. **Cortar fluff** — remover hesitações, "éé...", silêncios >0.5s
5. **Adicionar legendas burned-in** — fonte grande, legível, contraste alto, centralizada
6. **Formatação 9:16** — crop inteligente ou blur de background se necessário
7. **Design de loop** — ajustar final para conectar ao início se possível

### Especificações Técnicas
- Resolução: 1080×1920 (9:16)
- FPS: 30fps mínimo
- Áudio: normalizar a -14 LUFS
- Legendas: fonte Arial Bold ou similar, tamanho 72–80pt, sombra para contraste
- Duração alvo: 15–45 segundos (máximo de completion rate)

## Framework de Copywriting Multi-Plataforma (Paula Post)

### Estrutura Universal de Caption
```
[HOOK — primeiras 125 chars / antes do "ver mais"]
↓
[CORPO — contexto e valor adicional]
↓
[CTA — ação específica]
↓
[HASHTAGS — específicas por plataforma]
```

### CTAs de Alta Conversão por Objetivo
| Objetivo | CTA |
|---------|-----|
| Compartilhamento | "Manda pra quem precisa ver isso" |
| Salvamento | "Salva pra não perder depois" |
| Comentário | "Você sabia disso? Comenta aí" |
| Seguidor | "Sigo pra mais assim toda semana" |
| Afiliado | "Link na bio pra [benefício direto]" |

### Regras por Plataforma
- **TikTok**: Keywords no caption (é mecanismo de busca). Máx. 5 hashtags de nicho.
- **Reels**: Front-load do hook em 125 chars. DM-share bait. Trending audio quando possível.
- **YouTube Shorts**: Inclua #Shorts. Título curto e baseado em curiosidade.
- **Kwai**: Caption simples e direto. Emojis estratégicos.
- **X**: Max 280 chars com o clip. Uma frase-gancho forte. 1–2 hashtags.
- **Threads**: Contexto em texto + clip. Mais conversacional que outras plataformas.

## Framework de Monetização (Mônica Monetiza)

### Avaliação por Fase
```
Fase 1 (0–10k): Afiliado imediato via bio
Fase 2 (10k–100k): Programas + patrocínio
Fase 3 (100k+): Produto próprio + patrocínio direto
```

### Ações Imediatas por Vídeo
1. Identificar se o nicho do vídeo tem afiliado de alta conversão
2. Criar CTA de afiliado específico para esse vídeo
3. Atualizar link na bio se necessário
4. Registrar no tracker de performance
