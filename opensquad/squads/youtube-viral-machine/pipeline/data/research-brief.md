# Research Brief — YouTube Viral Machine

## Domínios Pesquisados

1. Psicologia de atenção em vídeo curto (short-form)
2. Algoritmos do TikTok, Instagram Reels, YouTube Shorts
3. Técnicas de edição para retenção de audiência
4. Estratégias de monetização multi-plataforma (mercado BR)

## 1. Psicologia de Atenção — Fundamentos Científicos

### A Janela dos 3 Segundos
- Estudo do Facebook: 65% das pessoas que assistem os primeiros 3 segundos continuam por 10+ segundos; 45% chegam a 30+ segundos.
- Cerca de 50% dos espectadores abandonam nos primeiros 3 segundos.
- Reels com hold rate >60% nos primeiros 3 segundos alcançam 5–10× mais pessoas que Reels com hold rate <40%.

### Interrupção de Padrão (Pattern Interrupt)
- O cérebro humano é treinado a reconhecer e ignorar padrões repetitivos.
- Quando algo quebra o padrão, o reflexo de atenção é ativado automaticamente.
- Recomendação: quebrar o padrão visual ou auditivo a cada 5–7 segundos dentro do vídeo.

### O Gap de Curiosidade (Curiosity Gap)
- A informação parcialmente revelada cria tensão cognitiva que só se resolve assistindo até o fim.
- Estudos da HubSpot (2024): curiosity gaps aumentam o watch time em até 20%.
- Mecanismo: o cérebro é compelido a "fechar o loop" — informação incompleta é desconfortável.

### Atenção ao Rosto Humano (Face Bias)
- Pesquisa da ScienceDirect (Koh & Cui, 2022): thumbnails com número moderado de rostos têm desempenho superior a thumbnails sem rostos ou com muitos rostos (curva U invertida).
- Rostos com expressões emocionais intensas (surpresa, felicidade, choque) aumentam CTR (taxa de clique).
- Hack validado: overlay de rosto de pessoa conhecida/expressivo no canto do vídeo captura atenção extra via "contagion emocional" (teoria do contágio emocional — Hatfield et al.).
- Plataformas com algoritmo de recomendação (YouTube Shorts) favorecem thumbnails emocionalmente carregadas (pesquisa: Algorithmic Bias in YouTube Shorts, arXiv 2025).

### Arousal Emocional e Compartilhamento
- Conteúdo que gera emoções de alta excitação (surpresa, espanto, raiva, inspiração intensa) é compartilhado mais do que conteúdo neutro.
- Teoria do arousal emocional (Berger & Milkman): emoções de alta excitação ativam o impulso de compartilhar.

## 2. Algoritmos das Plataformas

### Métricas Universais de Ranqueamento
| Métrica | Peso |
|---------|------|
| Watch time % | ★★★★★ |
| Replays / loops | ★★★★★ |
| Shares via DM | ★★★★★ |
| Saves | ★★★★ |
| Comments | ★★★ |
| Likes | ★★ |

### Instagram Reels
- Shares via DM são o sinal mais forte para distribuição.
- "Originality Score": conteúdo com marca d'água do TikTok perde alcance.
- Trial Reels: teste com não-seguidores antes de publicar para todos.
- Keywords em captions > hashtags para descoberta (2025).
- Trending audio: janela ótima = 2–7 dias após o som viralizar.

### TikTok
- Tornou-se mecanismo de busca para a Geração Z — keywords em captions são indexadas.
- Algoritmo favorece nichos específicos (#BookTok, #FinanceTok, etc.) e amplifica dentro do nicho.
- Loop seamless = replay = sinal de qualidade.

### YouTube Shorts
- Inclua #Shorts no título ou descrição para categorização correta.
- Feed de Shorts é separado do algoritmo de vídeos longos.
- Replays são o sinal primário — design de loop é crítico.

### Kwai
- Segue padrão semelhante ao TikTok; recompensa conteúdo original e consistência.
- Creator Program disponível para contas BR com mínimo de seguidores.

### X (Twitter) e Threads
- X: vídeos nativos têm muito mais alcance que links externos. Clips de 30–60s funcionam bem.
- Threads: formato complementar ao Reels — use para contextualizar o clip em texto.

## 3. Técnicas de Edição para Máxima Retenção

### Loop Design
- O final deve levar naturalmente de volta ao começo (loop seamless).
- Técnicas: terminar com uma pergunta que o início responde; match cut visual; narração circular.

### Ritmo de Cortes
- Troque de cena ou ângulo a cada 2–3 segundos para manter interesse visual.
- Vídeos de 15–30 segundos têm os maiores completion rates.

### Legendas Burned-In (Obrigatório)
- 85% dos usuários de redes sociais assistem vídeos sem som.
- Legendas burned-in (gravadas no vídeo) aumentam watch time e acessibilidade.

### Proporção 9:16
- Nunca recortar horizontal para vertical — resultado parece amador e tem desempenho inferior.
- Deve ser gravado/editado nativamente em vertical.

### Overlay de Rosto
- Adicionar imagem/clip de rosto expressivo no canto do vídeo (picture-in-picture) aumenta atenção.
- Funciona por viés cognitivo: o cérebro prioriza rostos automaticamente (face detection neural pathway).

## 4. Monetização Multi-Plataforma (Mercado BR)

### Programas Nativos de Plataforma
| Plataforma | Programa | Requisito Mínimo |
|-----------|---------|-----------------|
| YouTube | YouTube Partner Program | 1.000 sub + 4.000h ou 10M Shorts views |
| TikTok | TikTok Creator Rewards | 10.000 seguidores + 100.000 views/30d |
| Instagram | Reels Bonus (Meta) | Convite da Meta (varia por conta) |
| Kwai | Kwai Creator Program | Varia por região |

### Afiliados (Alta Conversão no BR)
- **Produtos digitais**: Hotmart, Monetizze, Eduzz — comissões de 30–80%
- **Fintechs**: Nubank, C6 Bank, Inter, Nomad — CPA de R$50–R$150 por conta aberta
- **Cursos online**: Udemy (10–45%), Coursera
- **Amazon Afiliados**: produtos físicos via link na bio

### Funil de Monetização Rápida
1. **Fase 1 (0–10k seguidores)**: Afiliados via link na bio / swipe up
2. **Fase 2 (10k–100k)**: Programas de criadores + patrocínios menores
3. **Fase 3 (100k+)**: Patrocínios diretos + produto próprio (infoproduto)

### Estratégia de Link na Bio
- Use Linktree, Stan.store ou Bio.link para centralizar todos os links de afiliado.
- CTA direto em cada vídeo: "Link na bio para [benefício específico]".
