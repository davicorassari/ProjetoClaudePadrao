# Critérios de Qualidade — YouTube Viral Machine

## Critérios de Qualidade: Seleção de Momentos (Vitor Viral)

| Critério | Threshold Mínimo |
|---------|-----------------|
| Score viral (fórmula do framework) | ≥ 0.65 |
| Duração do clipe | 15–60 segundos |
| Clareza sem contexto anterior | O clip faz sentido sozinho |
| Energia do speaker | Detectável (variação de tom, ritmo) |
| Ausência de jargão técnico excessivo | Compreensível para audiência geral |

**Critério de veto:** Qualquer clipe que depende de contexto anterior para fazer sentido deve ser descartado automaticamente.

---

## Critérios de Qualidade: Edição (Carlos Corte)

| Critério | Threshold |
|---------|---------|
| Resolução | 1080×1920 mínimo |
| Proporção | 9:16 exato |
| Duração | 15–45 segundos |
| Subtítulos | Presentes em 100% dos frames com fala |
| Legibilidade dos subtítulos | Fonte ≥72pt, contraste WCAG AA |
| Silêncios removidos | Pausas >0.5s cortadas |
| Marca d'água de outras plataformas | Zero tolerância |
| Áudio normalizado | -14 LUFS ±2 |

**Critério de veto:** Clip sem legenda = reprovado automaticamente (85% assiste sem som).

---

## Critérios de Qualidade: Captions (Paula Post)

| Critério | Threshold |
|---------|---------|
| Hook nos primeiros 125 chars | Obrigatório para Instagram/Threads |
| CTA presente | Obrigatório em todos |
| CTA específico (não genérico) | "Comenta X" > "Deixa um comentário" |
| Keywords de nicho no caption | Obrigatório para TikTok |
| #Shorts no título/caption | Obrigatório para YouTube |
| Hashtags relevantes | 3–5 para TikTok, 5–15 para Reels |
| Tom adequado à plataforma | Adaptado por plataforma |

**Critério de veto:** Caption genérico (mesmo texto em todas as plataformas sem adaptação) = reprovado.

## Critérios de Qualidade: Publicação (Roberto Reach)

| Critério | Threshold |
|---------|---------|
| Sem marca d'água de plataforma concorrente | Obrigatório |
| Arquivo enviado nativamente (não como link) | Obrigatório |
| Caption correto publicado junto | Obrigatório |
| Hashtags incluídas | Obrigatório |

## Critérios de Qualidade: Monetização (Mônica Monetiza)

| Critério | Threshold |
|---------|---------|
| Afiliado identificado para o nicho | Mínimo 1 por execução |
| Link de afiliado ativo e válido | Verificado |
| CTA de afiliado incluído no relatório | Obrigatório |
| Próximos passos de monetização listados | Mínimo 3 ações |
| Fase de monetização identificada | Obrigatório |
